Welcome to BioJSON community site.

BioJSON was introduced in Jalview 2.9 to serve as a standard JavaScript Object Notation (JSON) model for representing/ exchanging Bioinformatics alignment data. This is the first version of the specification and we welcome all feedbacks to make BioJSON more standardised, sustainable and widely adopted worldwide.

BioJSON Vision

Our vision is to arrive at a point where any alignment data represented in JSON conforms to the BioJSON specifications.

Advantages of JSON/BioJSON

  • BioJSON can represent very sophisticated sequence alignments along with their linked-data better than most Bioinformatics data format
  • BioJSON can be easily embedded in HTML pages and some image formats
  • BioJSON is based on JSON which is very popular and can be parsed in almost all programming languages
  • Integrates seamlessly with the HTTP rest architecture
  • Very light weight, self describing and easy to understand
  • JSON format is text only and uses JavaScript syntax
  • JSON is hierarchical (i.e. it can contain values within values)

More information about JSON can be found at http://json.org/.

BioJSON can currently represent any sequence alignment data along with its annotations, features, groups, hidden sequences, hidden columns, colour schemes, and custom application settings. However phylogenetic trees, structures and more sophisticated annotations are not yet supported but will gradually be introduced in future versions of BioJSON.

The BioJSON Schema

  • When the mouse is hovered over the elements in the BioJSON schema section, traversable elements become highlighted and clickable
  • Click the “object” button or the plus “+” link to expand an object
  • Click the minus “-” to collapse an object
  • Click the parenthesis “{}” link to view the JSON schema which provides a more technical description of the data
  • When you expand an object the following information becomes visible: The variables in the object, their data types and the description of each variable. This exactly corresponds to how it appears in BioJSON

See BioJSON in Action!

The current application of BioJSON in Jalview 2.9 includes:
  • Export/ import of alignment to/from a JSON flat file
  • Embedding an alignment in HTML (File → Export Image → HTML)
  • Embedding an alignment in BioJS MSA viewer (File → Export Image → BioJS)
  • Retrieval of embedded alignment data from HTML/BioJS files (Drag and drop file in Jalview)
Future applications of BioJSON in Jalview:
  • Primary data protocol for Jalview web services (JalviewWS)
  • Primary data protocol for the Javascript version of Jalview (JalviewJS)
  • Storage of linked alignments (i.e. CDNA/Protein alignments)
