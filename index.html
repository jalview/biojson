Welcome to BioJSON community site.

BioJSON is a JavaScript Object Notation (JSON) standard for the representation and exchange of bioinformatics data. The first version of the specification was developed in 2015 and implemented in Jalview 2.9.

Get involved

Our aim with BioJSON is to allow web based and standalone bioinformatics applications to efficiently and easily exchange data and results, enabling tools to be more sustainable and widely adopted worldwide. If you have any suggestions or want to contribute to the BioJSON standard, then get in contact or file an issue.

Why use BioJSON

  • BioJSON v1 represents multiple sequence alignments, quantitative and symbolic sequence and alignment annotation, and also allows rendering and display styles to be encoded
  • BioJSON can be embedded as a supporting data for interactive web pages
  • JSON is very popular and can be parsed in almost all programming languages
  • Integrates seamlessly with the HTTP rest architecture
  • Very light weight, self describing and easy to understand

We think that BioJSON is the best choice for representing alignment and associated data on the web. It provides a standard way of handling multiple alignments and annotations, which enables interoperation between stand-alone and web based multiple alignment and analysis systems.

What does BioJSON support?

BioJSON v1 is designed to represent multiple sequence alignments along with their annotations, features, groups, hidden sequences, hidden columns and colour schemes. It also allows custom application settings to be encoded for the Jalview multiple alignment workbench.

Future versions of BioJSON will provide more complete support for interchange of multiple alignment data, and introduce new representations for phylogenetic data and 3D structures.

The BioJSON Schema

BioJSON is specified using version 4 of the jsonschema standard. The schema is available via the github repository or can be downloaded from here. You can explore the BioJSON schema with the interactive visualisation created with Docson.

Instructions for interacting with the Docson schema

  • When the mouse is hovered over the elements in the BioJSON schema section, traversable elements become highlighted and clickable
  • Click an object’s “object” button to expand it, or the plus “+” link to expand all objects
  • Click the minus “-” to collapse all objects
  • The parenthesis “{}” link shows the underlying JSON schema which provides a formal description of the BioJSON format
  • Expanding an object reveals its variables, their type and a brief description in the object, their data types and the description of each variable.

See BioJSON in Action!

The current application of BioJSON in Jalview 2.9 includes:
  • Export and import of BioJSON data as JSON flat files
  • Export and import of supporting data for interactive web pages, including:
    • Interactive SVG figures (File → Export Image → HTML)
    • The BioJS MSA viewer (File → Export Image → BioJS)

Future capabilities

Some of our goals for future versions of BioJSON are:
  • BioJSON services for Jalview desktop
  • BioJSON for the Javascript version of Jalview (JalviewJS)
  • Representation of linked CDNA and Protein alignments views
